python-setuptools

Interface module to the python setuptools for building and distributing python packages

SYNOPSIS

rerun [ARGS] python-setuptools: [ build ] [OPTIONS]

README

python-setuptools, is an interface for managing Python PyPI distributions.

This python-setuptools module will generate a system package (e.g. RPM) based on a versioned PyPI archive.

The system package generated by this module may then be installed by the native system packaging tool instead of tools like easy_install or pip.

  • Use python-setuptools:build to generate an RPM based on a versioned PyPI distribution package.

Consult README.md in the command and options directories for further description of using these commands.

Requirements

  • Internet access in order for the module to download archives from PyPI.
  • A Red Hat/CentOS 5 or 6 instance

COMMANDS

  • build: Produce a package of indicated format based on a PyPI based distribution

OPTIONS

  • directory: Directory location to produce the rpm
  • format: output packaging format to produce
  • name: name of the PyPI archive to package
  • release: The package release
  • url: base url to the python PyPI distribution archives
  • version: version of the PyPI distribution to package
